## VAR - LET - CONST
* var 
  * chỉ có var mới vừa khai báo và cập nhật lại giá trị của biến
  * hoisting
  * khai báo bên trong function => function/locally scoped, ngược lại globally scoped
* let
  * không thể khai báo lại, nhưng có thế cập nhật lại giá trị
  * hoisting
  * block scoped
* const
  * block scoped: block scope và hoisting
  * kiểu primitive (string, number, boolean, null, undefined) => không thế tái khai báo hay cập nhật lại giá trị mới cho biến
  * kiểu reference (object, array, function) => không thể khai báo hay cập nhật lại giá trị mới cho biến nhưng có thể cập nhật lại giá trị thuộc tính của biến     
## DATA TYPES
* primitive: string, number, boolean, null, undefined
* reference: object, array, function

### STRING TEMPLATE LITERALS
* Dùng back-ticks (\` \`) để
  * Quotes inside string: 
    ```
    let text = `He's often called "Johnny"`;
    ```
  * Multiline strings
    ```
    let text =
    `The quick
    brown fox
    jumps over
    the lazy dog`;
    ```
* Interpolation (nội suy)
  ```
  let firstName = "John";
  let lastName = "Doe";

  let text = `Welcome ${firstName}, ${lastName}!`;
  ```
  ```
  let price = 10;
  let VAT = 0.25;

  let total = `Total: ${(price * (1 + VAT)).toFixed(2)}`;
  ```

### ARRAYS METHODS
* toString(), join(","): convert to String
* pop(): remove item cuối cùng
* push(): thêm item vào cuối cùng
* shift(): remove item đầu tiên, index bị giảm
* unshuift(: thêm item vào đầu
* delete: để lại một chỗ trống underfined trong mảng
* splice(): nối thêm item, hay remove 
  ```
  const fruits = ["Banana", "Orange", "Apple", "Mango"];
  fruits.splice(2, 0, "Lemon", "Kiwi");
  const fruits = ["Banana", "Orange", "Apple", "Mango"];
  fruits.splice(0, 1);
  ```
* concat(): merge mảng
  ```
  const myGirls = ["Cecilie", "Lone"];
  const myBoys = ["Emil", "Tobias", "Linus"];

  const myChildren = myGirls.concat(myBoys);
  ```
* slice(): cắt mảng => tạo ra một array mới
  ```
  const fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"];
  const citrus = fruits.slice(1);
  ```

* map: tạo ra mảng mới bằng cách thực hiện function trên mỗi phần tử của mảng, không thay đổi mảng gốc

  }

  ```
  const numbers = [45, 4, 9, 16, 25];

  function myFunction(value, index, array) {
      return value > 18;
  }
  ```
