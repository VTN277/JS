## Intro
* khi một trang web được tải, trình duyệt sẽ tạo ra DOM [Doument Object Model(mô hình đối tượng tài liệu)] của trang 
* The HTML DOM Tree of Objects 
* ![image](https://user-images.githubusercontent.com/67737894/143903545-0dd8676e-855a-4c14-99cf-3ee74ca947df.png)
* DOM chuẩn W3C được chia thành 3 thành phần khác nhau
  * Core DOM - mô hình tiêu chuẩn cho các loại tài liệu
  * XML DOM - mô hình tiêu chuẩn cho tài liệu xml
  * HTML DOM - mô hình tiêu chuẩn cho các tài liệu HMTL
* HTML DOM: là tiêu chuẩn về cách get, change, add hay delete HTML elements
## HTML DOM Methods
* Trong DOM, tất các các HTML elements dều được định nghĩa là đối tượng
* A **property** là giá trị mà bạn có thể get or set (like changing content of an HTML element)
* A **method** là hành động mà bạn có thể làm (like add or deleting an HTML elemnent)
  ```
  <html>
  <body>

  <p id="demo"></p>

  <script>
  document.getElementById("demo").innerHTML = "Hello World!";
  </script>

  </body>
  </html>
  //In the example above, getElementById is a method, while innerHTML is a property.
  //innerHTML hữu dụng để thay thế nội dung phần tử của HTML elemenns
  ```
