## 2.2 Code Structure
### Statements
* Ngăn cách bới dấu chấm phấy
### Semicolons 
* Implicit semicolon: 
```
alert('Hello')
alert('World')
```
* Not Automatic Semicolon
```
alert("Hello");

[1, 2].forEach(alert);
// result: Hello, 1, 2
```
```
alert("Hello")

[1, 2].forEach(alert);
//result: Hello, error
//Bởi vì JS không có dấu chấm phấy trước dâu ngoặc vuông nên đây là một câu lệnh duy nhât 
```
## 2.4 Variable
* Quy tắc: biến chứa chữ, số, $, _ , không bắt đầu bởi số
## 2.6 Interaction: alert, prompt, confirm
* alert: show message
* prompt: message + input text
* confirm: message + option (OK, cancel)
