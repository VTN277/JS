## HelloWord
* Type mặc định trong thẻ script: type="text/javascript" nếu không khai báo
## Code Structure
### Statements
* Ngăn cách bới dấu chấm phấy
### Semicolons 
* Implicit semicolon: 
```
alert('Hello')
alert('World')
```
* Not Automatic Semicolon
```
alert("Hello");

[1, 2].forEach(alert);
// result: Hello, 1, 2
```
```
alert("Hello")

[1, 2].forEach(alert);
//result: Hello, error
//Bởi vì JS không có dấu chấm phấy trước dâu ngoặc vuông nên đây là một câu lệnh duy nhât 
```
